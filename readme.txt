Enables easy search of email addresses (wrapped with and without mailto in anchor tag) from all posts, custom posts, pages and comments in the website.

== Description ==

Users can search for email address that are wrapped with mailto (clickable) and without mailto (unclickable) in anchor tags from all over the website in fraction of time. It is easy to find out the email addresses from all over the wordpress website and also identify which of those are the clickable and not clickable. One can easily find out the source of the email addresses i.e in which pages or posts the email addresses are present.

Hence, the task of finding out email addresses without mailto in anchor tag in the WordPress website will be a very easy task. Admin can easily locate the unclickable email addresses in the website and their sources (posts, pages) and replace them to become clickable with mailto in anchor tag.

= Features =

* Find out the email addresses from the entire website (posts, pages and comments) in least possible time
* Find out the email addresses from "Post Content", "Post Excerpt", and "Comment Content" fields of posts, pages, and comments.
* Identify the clickable and unclickable email addresses
* Identify the source of the particular email addresses that are listed out
* Sort out the email address listing based on the date on which the particular post, page or comment was published

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Search Email' screen to search email address.
